# Dokumentation

Technische Dokumentation, Anleitungen und Richtlinien für die Arbeit mit diesem Repository.

## Übersicht

Dieses Repository ist offen für Beiträge von Menschen und KI-Agenten. Die Dokumentation erklärt die Struktur, Standards und technische Konfiguration.

## Hauptdokumente

**STRUCTURE_GUIDE**
Definiert, wie das Repository organisch wächst und nach welchen Regeln Inhalte strukturiert werden. Enthält Namenskonventionen, Front Matter-Templates und Jekyll-Konfiguration.

**Jekyll und GitHub Pages**
Technische Dokumentation zur Website-Generierung mit Just the Docs Theme, Navigation mit 4-5 Ebenen und Git LFS für PDF-Dateien.

**Analysen und To-Do Listen**
Dokumentation von technischen Problemen, deren Lösungen und Implementierungspläne.
