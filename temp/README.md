# Temp - Temporärer Arbeitsbereich

Dieser Ordner dient für:
- Work-in-Progress TODO-Listen
- Debugging-Dateien
- Experimentelle Features
- Temporäre Tests

## Wichtig

- Dieser Ordner ist in Jekyll `_config.yml` excluded (nicht auf Website sichtbar)
- Dateien hier werden **versioniert** (in Git committed)
- Nach Abschluss einer Aufgabe: Dateien ins `archive/` verschieben oder löschen

## Workflow

1. **Während der Arbeit:** Dateien hier ablegen
2. **Nach Abschluss:**
   - Relevante Dateien → `archive/YYYY-MM-DD/`
   - Nicht mehr benötigte Dateien → löschen
3. **Dokumentieren:** Archivierung im `archive/ARCHIVE_LOG.md` vermerken

## Siehe auch

- `/archive/` - Archivierte abgeschlossene Arbeiten
- `archive/ARCHIVE_LOG.md` - Log aller Archivierungen
